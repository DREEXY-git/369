# Phase 2-B-5 実装 — AIが営業プレイブックを読めるようになった

> 出典: `369` リポジトリ `docs/audit/60_phase2b5_sales_playbook_ai_reference_impl.md`（判定 GO・commit-only）。
> 関連: [[Phase2B5SalesPlaybookAI参照設計]] / [[Phase2B4本番確認]] / [[Phase2A3c2CompanyBrainAI参照]] / [[PhaseX04本番スモーク定型化]]

## 何ができるようになったか

- **ナレッジ検索で質問すると、AIが営業プレイブック（売り方の型）も回答の材料に使う**ようになった。参照したプレイブックは「参照した会社の頭脳」にタイトルで表示される。
- これで会社の頭脳3テーブル（会社方針・商品カタログ・営業プレイブック）すべてが「人間が書き・AIが読む」体制になった（ローカル検証まで。**本番確認はこれから**）。

## 安全の型（設計 [[Phase2B5SalesPlaybookAI参照設計]] どおり実装）

- **読むだけ**: AIは営業プレイブックを書き換えない・消せない（書き込みは人間専用のまま）。
- **自テナントのみ・アーカイブ済み除外・機密ラベルは通常/社内限のみ・高機密は読まない**。
- **読んだら記録**: どのプレイブックを参照したかが監査ログ（ai_reference）にレコード単位で残る。
- **外部AIには出ない**: 全件「外部AI送信禁止」のまま＋許可UIなし＝外部LLM時は構造的にゼロ。
- 「言わないこと」（No.1表記・効果保証などの禁止例）は、AIが肯定文として引用しないよう**「言わない: 〜」の明示付き**で渡す。
- 一度に読む上限は従来どおり**合計5件**（増やしていない）。
